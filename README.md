# YouTube Speed Controller

A Chrome extension that gives you more control over YouTube playback speed.

## Features
-   **Default 1.5X Speed**: Automatically sets the video speed to 1.5X when a YouTube video loads.
-   **Toggle Control**: Easily switch between standard (1X) and fast (1.5X) speed using an on-screen slider toggle.
-   **Modern UI**: Clean, 3D-styled control panel that blends with the interface.

## Installation from Source
1.  Clone or download this repository.
2.  Open Chrome and go to `chrome://extensions/`.
3.  Enable **"Developer mode"** in the top right.
4.  Click **"Load unpacked"**.
5.  Select the folder where you downloaded this code.
